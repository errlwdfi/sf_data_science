## Goals of the project

* Predict GDP values for the last 3 years.

* Interpolate missing data.

* Predict GDP volatility for the last 3 years.


## Data

Feature description:

* *year* - year of observation (as index).

* *GDP (current US$)* - GDP in current USD.
